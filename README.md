# Escopo do Projeto:

Será desenvolvido um portal Web e app mobile nas plataformas Android e iOS visando a atualização cadastral dos colaborares de uma organização para entrega do e-Social;

Os cadastros a serem revisados e atualizados deverão ser enviados pela organização empregadora para a portal contendo obrigatoriamente o e-mail, perfil do facebook ou número de celular do colaborador;

O colaborador será notificado pelo contato informado para realizar a atualização cadastral. Esta atualização poderá ser realizada via portal Web ou pelos aplicativos mobile;

O usuário ( colaborador ) não poderá incluir um novo cadastro caso não seja pré-enviado por uma organização empregadora;

O usuário ( colaborador ) somente pode visualizar e editar seu próprio cadastro. O mesmo não terá acesso a demais cadastros;

A usuário ( organização empregadora ) somente terá acesso aos dados atualizados dos quais solicitou. Uma organização não poderá visualizar ou editar registros de colaboradores de outra organização;

Os dados deverão ser enviados pela organização empregadora em planilha formato XLS com layout a ser definido neste escopo;


# Restrições do Escopo:

O sistema denominado de Meu Cadastro eSocial não haverá integração com a Receita Federal ou quaisquer outro órgão relacionado ao projeto eSocial;

O sistema também não irá qualificar as informações com quaisquer que sejam órgão competente do projeto eSocial;

Toda a validação e qualificação das informações obtidas pelo sistema deverão ser realizadas pelo sistema de gestão de Recursos Humanos da organização empregadora;

O sistema apenas exportará um planilha no formato XLS com os dados, não haverá integração com o sistema de gestão de Recursos Humanos da organização empregadora.
